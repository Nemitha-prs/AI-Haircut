  QUICK GUIDE: Setting Up OpenAI API Key
═══════════════════════════════════════════════════════════════

STEP 1: Get Your OpenAI API Key
───────────────────────────────────────────────────────────────

1. Go to: https://platform.openai.com/api-keys
2. Sign in to your OpenAI account
3. Click "Create new secret key"
4. Give it a name (optional): "Haircut AI App"
5. Click "Create secret key"
6. ⚠️ COPY THE KEY IMMEDIATELY - you won't see it again!
7. The key will look like: sk-abc123def456... (50+ characters)

STEP 2: Add Key to .env File
───────────────────────────────────────────────────────────────

1. Open: backend/.env
2. Add this line (replace with your actual key):

   OPENAI_API_KEY=sk-your-complete-key-here

⚠️ IMPORTANT FORMAT:
   • NO quotes around the key
   • NO spaces before or after the = sign
   • Key must be on ONE line (don't press Enter)
   • Copy the ENTIRE key

STEP 3: Save and Test
───────────────────────────────────────────────────────────────

1. Save the .env file (Ctrl+S)
2. Test your key:

   cd backend
   node test-key.js

3. You should see: ✅ SUCCESS! Your API key is working!

STEP 4: Start Server
───────────────────────────────────────────────────────────────

   cd backend
   npm start

═══════════════════════════════════════════════════════════════

EXAMPLE .env FILE:
───────────────────────────────────────────────────────────────

OPENAI_API_KEY=sk-abc123def456ghi789jkl012mno345pqr678stu901vwx234yz567
PORT=3000

═══════════════════════════════════════════════════════════════

COMMON MISTAKES:
───────────────────────────────────────────────────────────────

❌ WRONG: OPENAI_API_KEY = "sk-..."  (has spaces and quotes)
❌ WRONG: OPENAI_API_KEY=sk-abc      (key is incomplete)
❌ WRONG: OPENAI_API_KEY=sk-abc      (key split across lines)
          xyz789

✅ CORRECT: OPENAI_API_KEY=sk-YourCompleteKeyHereOnOneLine

═══════════════════════════════════════════════════════════════

TROUBLESHOOTING:
───────────────────────────────────────────────────────────────

If you get "401 Authentication failed":
1. Make sure key is complete (not cut off)
2. Check no quotes/spaces in .env file
3. Create a NEW key in OpenAI dashboard
4. Restart server after changing .env
